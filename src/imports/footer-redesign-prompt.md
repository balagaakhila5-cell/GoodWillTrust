Modern Footer Redesign Prompt

Prompt:

Redesign the footer section of the “Good Will Trust – Empowering Abilities” NGO website to make it modern, visually appealing, and professional while maintaining the white and blue color theme of the website.

Footer Background

Set the footer background color to pure white (#FFFFFF).

Use soft blue accents and dividers to maintain the NGO branding.

Add a subtle top border or gradient line using the primary blue color.

Footer Layout

Create a clean 4-column responsive footer layout:

Column 1 – About the NGO

Small description about Good Will Trust

Focus on empowering people with disabilities and community support

Column 2 – Quick Links

Home

About Us

Programs

Donate

Volunteer

Contact

Column 3 – Contact Information

Address

Phone number

Email

Working hours

Column 4 – Follow Us

Social media icons (Facebook, Instagram, Twitter, LinkedIn)

Icons should have hover animations

Modern UI Design

Use rounded cards or sections

Add soft shadows

Maintain good spacing and alignment

Use clean typography (Poppins / Montserrat for headings, Open Sans / Roboto for body text)

Animations

Add subtle modern animations:

Hover animation on links

Text color changes to blue

Smooth underline animation

Social media icons

Slight scale-up on hover

Soft glow or background circle animation

Fade-in animation when the footer enters the viewport

Smooth micro-interactions for buttons and links

Newsletter Section

Add a newsletter subscription area above the footer:

Heading: Stay Connected

Short text encouraging users to subscribe for updates

Email input field

Subscribe button with blue gradient

Add a button hover animation

Bottom Footer Bar

At the very bottom include:

Copyright text
© 2026 Good Will Trust. All Rights Reserved.

Small divider line above it

Optional Back to Top button on the right side with smooth scroll animation

Responsiveness

On tablet, convert to 2 columns

On mobile, stack into single column layout

Ensure touch-friendly spacing

Design Goal

The footer should feel:

Modern

Trustworthy

Accessible

NGO-professional

Clean and elegant
